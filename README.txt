# DrumTune PWA package

Files included:
- index.html
- manifest.json
- service-worker.js
- icons/

To use:
1. Upload the contents of this folder to any HTTPS host (GitHub Pages, Netlify, Vercel, etc.).
2. Open the deployed URL in Safari on iPhone.
3. Allow microphone access.
4. Use Share > Add to Home Screen.

Notes:
- Microphone access usually requires HTTPS.
- Opening local HTML directly from Files on iPhone is not reliable for this kind of app.
